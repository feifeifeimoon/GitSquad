# GitSquad - Your autonomous developer team on GitHub.

![GitSquad banner](docs/assets/banner.png)

**GitSquad** coordinates multiple specialized AI agents to plan, execute, and verify code changes, enabling high-performance collaborative software development directly within your repositories.

## Overview

**GitSquad** is a multi-agent orchestration framework for autonomous software development workflows on GitHub. Instead of relying on a single general-purpose assistant, **GitSquad** organizes a team of specialized agents that can collaborate across the full development loop: understanding an issue, proposing a plan, editing code, reviewing changes, and validating the result.
